Frequently asked questions

Clear expectations. No mystery package theater.

Is this a subscription or a one-time install?▾

These packages are one-time installs. You are paying to have the delivery system set up and configured around your workflow.

Do I keep what gets built?▾

Yes. Your workspace, links, and installed structure remain yours. This is not rented access to some disposable portal.

Which package should most businesses start with?▾

Pro is usually the best starting point for active service businesses because it adds lifecycle protection, uploads, and templated delivery control without jumping straight into more advanced routing.

Do clients need logins to use the system?▾

No. The system is designed around simple links for intake, payments, uploads, and status visibility so the client experience stays easy while your internal operation stays structured.

Can I start smaller and upgrade later?▾

Yes. That is the sane way to do it. Start with the install your current workload actually needs, then layer deeper automation when the added complexity is justified.
